리액트의 내부 상태관리
- 내부 상태 변경시 리액트는 변경된 해당 컴포넌트 함수를 호출함. -> 가상의 DOM 요소를 생성해 기존의 DOM과 비교 -> 변경된 부분만 업데이트함.
- 리액트에서 변경 가능한 상태는 useState를 사용해야 함.
